Conti_Ransomware

Overview#

This Threat Sighting documents Conti ransomware TTPs based on Conti leaks (back in August 2021 and more recently Feb/March 2022). This Threat Sighting is based on public reports.

This document is a summary of the extracted TTPs. For full details please refer to the AC3 Threat Sighting (.yml file) at: https://github.com/mcafee-enterprise/ac3-threat-sightings/blob/main/sightings/Sightings_Conti_Ransomware.yml

Author: Alejandro Houspanossian (@lekz86)

Acknowledgements: @TheDFIRReport, @vxunderground

Threat Actor Activity#

  • Conti used WMIC via Cobalt Strike for Execution.
  • Conti used vssadmin via WMI and Cobalt Strike to create a Volume Shadow Copy for Collection.
  • Conti used doscmd/copy command via WMI and Cobalt Strike to move system files including NTDS.dit for Collection.
  • Conti used 7zip via WMI and Cobalt Strike to archive collected files for Collection.
  • Conti used ntdsutil for Credential Access.
  • Conti used NetScan.exe via Cobalt Strike to scan internal network for Discovery.
  • Conti used Rclone to upload data to cloud storage for Exfiltration.
  • Conti used Rubeus.exe via Cobalt Strike for Credential Access.
  • Conti used a PowerShell inline script via Cobalt Strike to download and install AnyDesk Remote Access Tool for Persistence.
  • Conti used PowerShell commandlets to allow incomming RDP connections on an uncommon port for Defense Evasion.
  • Conti used COM+ Service DLL via Cobalt Strike to dump LSASS process memory for Credential Access.
  • Conti used Task Manager over RDP to dump LSASS process memory for Credential Access.
  • Conti used WMIC via Cobalt Strike to launch comsvcs.dll via RunDll32 on remote systems for Credential Access.
  • Conti used PSEXEC via Cobalt Strike to launch comsvcs.dll via RunDll32 on remote systems for Credential Access.
  • Conti used PSEXEC via Cobalt Strike to copy malicious files into remote systems for Lateral Movement.
  • Conti used WMIC via Cobalt Strike to copy and start execution of malicious files into remote systems for Lateral Movement.
  • Conti used SCHTASKS via Cobalt Strike to execute malicious files into remote systems for Lateral Movement.
  • Conti used AdFind via Cobalt Strike for Discovery.
  • Conti used Cobalt Strike to launch whole domain encryption for Impact.
  • Conti used RDP to launch whole domain encryption for Impact.
