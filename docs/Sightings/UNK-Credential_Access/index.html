UNK-Credential_Access

Overview#

This Threat Sighting documents observed TTPs for Credential Access using COM+ Service DLL (comsvcs.dll) and a memory dumper (dumpert.dll). This sighting is based on direct observation.

This document is a summary of the extracted TTPs. For full details please refer to the AC3 Threat Sighting (.yml file) at: https://github.com/mcafee-enterprise/ac3-threat-sightings/blob/main/sightings/Sightings_UNK-Credential_Access.yml

Author: Alejandro Houspanossian (@lekz86)

Acknowledgements: @lekz86

References:#

  • ...

Threat Actor Activity#

  • Threat Actor used CMD to execute tasklist for Discovery.
  • Threat Actor used COM+ Service DLL via RunDll32 for dumping LSASS process memory for Credential Access.
  • Threat Actor used a Windows Service to launch RunDll32 via CMD for Persistence.
  • Threat Actor used dumpert.dll via RunDll32 for dumping LSASS process memory for Credential Access.
