TA551-Phishing

Overview#

This Threat Sightings documents TPPs of a TA551 campaign distributing Cobalt Strike.

This document is a summary of the extracted TTPs. For full details please refer to the AC3 Threat Sighting (.yml file) at: https://github.com/mcafee-enterprise/ac3-threat-sightings/blob/main/sightings/Sightings_TA551.yml

Threat Actor Activity#

  • Threat actor used Email with .zipped .doc file as attachment for Initial Access.
  • Threat actor used weaponized .doc files to lure users for Execution.
  • Threat actor used MSHTA to download a malicious .DLL masqueraded as a .JPG file for Command And Control.
  • Threat actor used REGSVR32 to launch payload for Defense Evasion.
  • Threat actor used REGSVR32 to download a masqueraded Cobalt Strike shellcode runner for Command And Control.
  • Threat actor used Cobalt Strike shellcode runner to execute a remote payload for Command And Control.
