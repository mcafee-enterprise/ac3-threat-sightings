Darkside_Ransomware

Overview#

This Threat Sighting documents observed TTPs for DarkSide ransomware. This Threat Sighting is based on direct observation.

This document is a summary of the extracted TTPs. For full details please refer to the AC3 Threat Sighting (.yml file) at: https://github.com/mcafee-enterprise/ac3-threat-sightings/blob/main/sightings/Sightings_Darkside_Ransomware.yml

Author: Alejandro Houspanossian (@lekz86)

Acknowledgements: @lekz86

References:#

  • https://www.mandiant.com/resources/shining-a-light-on-darkside-ransomware-operations
  • https://www.trellix.com/en-us/threat-center/threat-reports/jan-2022.html

Threat Actor Activity#

  • DarkSide used Valid Credentials over RDP for Initial Access.
  • DarkSide used Valid Credentials over RDP for Lateral Movement.
  • DarkSide used RDP to download software tools for Command And Control.
  • DarkSide used RDP to execute CMD for Execution.
  • DarkSide used CMD to execute a CVE-2021-27091 exploit for Privilege Escalation.
  • DarkSide used nltest.exe via interactive PowerShell for Discovery.
  • DarkSide used Cobalt Strike HTTP beacons for Command And Control.
  • DarkSide used Windows Services for Persistence.
  • DarkSide used Windows Services to launch Cobalt Strike beacon PE file for Command And Control.
  • DarkSide used process injection into RUNDLL32 and DLLHOST to launch post-exploitation tasks for Defense Evasion.
  • DarkSide used Cobalt Strike to execute BloodHound in memory for Discovery.
  • DarkSide used Windows Services to launch Cobalt Strike PowerShell command via CMD for Command And Control.
  • DarkSide used Cobalt Strike Named Pipes Impersonation for Privilege Escalation.
  • DarkSide used RDP to execute a renamed copy of rclone.exe for Exfiltration.
  • DarkSide used Rundll32 to launch encryption for Impact.
